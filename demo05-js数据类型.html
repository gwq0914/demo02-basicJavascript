数据类型,是什么，为什么必需

在计算机中，不同的数据所需占用的存储空间是不同的，为了便于把数据分成所需内存大小不同的数据，充分利用存储空间，于是定义了不同的数据类型。
数据类型就是数据的类别。

变量的数据类型

javascript是一种弱类型或者说动态语言。
这意味着不用提前声明变量的类型，在程序运行过程中，类型会被自动确定。赋值时才知道是什么数据类型
js是动态语言 变量的数据类型是可以变化的。

简单数据类型

简单数据类型 说明
Number 数字型，包含整型值和浮点型值，如21、0.21
Boolean 布尔值类型，如true、false,等价于1和0
String 字符串类型，如“张三”,在js里面，字符串都带引号
Undefined var a;声明了变量a但是没有给值，所以此时类型为undefined
Null var a=null;声明了变量a为空值。

数字型Number

数字类型既可以用来保存整数值，也可以保存小数值。
数字型中的特殊数字：
数字型 说明或注释 举例或公式
整数值 整数 var num=10;
小数值 小数 var PI=3.14;
八进制 程序里面数字前面加0 表示八进制 var num1=010;
十六进制 程序前面加0x 表示十六进制 var num2=0xa;
数字型的最大值 1.7976931348623157e+308 Number.MAX_VALUE
数字型的最小值 5e-324 Number.MIN_VALUE
数字型的无穷大 Infinity Number.MAX_VALUE*2
数字型的无穷小 -Infinity -Number.MAX_VALUE*2
非数字NaN 如果用加号连接，默认后面的数字为字符 '小星星'-5

字符串型String

字符串型可以是引号中的任意文本，其语法为双引号和单引号。
js更推荐使用单引号
字符串转义符
转义符 解释说明
\n 换行符号，n是newline的意思
\\ 斜杠\
\' 单引号
\" 双引号
\t tab缩进
\b 空格，b是blank的意思

弹出网页警示框

字符串的拼接:加号

字符串拼接的口诀：数值相加，字符相连吧
注意：变量不要写到字符串里面，是通过和字符串相连的方式实现的。 console.log('12'+12);
console.log(12+12);

字符串的案例：显示年龄的案例

弹出一个输入框，需要用户输入年龄，之后弹出一个警示框显示"您今年**岁了！"

布尔型Boolean

布尔类型有两个值：true和false，其中true表示真（1）（对），false表示假（0）（错）。

Undefined和Null

一个声明后没有被赋值的变量会有一个默认值undefined
var num;
console.log(num+1);//NaN
console.log(num+"是个数字！");//字符串型String
var kong=null;
console.log(kong+1);//1
console.log(kong+"是空值，计算时为0");//字符串型String
