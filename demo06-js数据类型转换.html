黑色是字符串
undefined 和 null 是浅灰色
晶蓝色 是 数字型

字面量

字面量是在源代码中一个固定值的表示法，通俗来说，就是字面量表示如何表达这个值。

数据类型转换

使用表单、prompt获取过来的数据默认是字符串型的，此时就不能直接简单的进行加法运算，而需要转换变量的数据类型。通俗的说，就是把一种数据类型的变量转换成另外一种数据类型。

转换为字符串类型

方式 说明 案例
toString()方法 转成字符串 var num=1;alert(num.toString);
String()强制转换 转成字符串 var num=1;alert(String(num));
加号拼接字符串 和字符串拼接的结果都是字符串，这种转换方式为隐式转换 var num=1;alert(num+"我是字符串");

转换为数字类型

方式 说明 案例
parseInt(String)函数 将字符型转换为整数数值型 parseInt('78')
parseFloat(String)函数 将字符型转换为浮点数数值型 parseFloat('78.5')
Number()强制转换 将字符型转换为数值型 Number('12')
使用-*/实现隐式转换 利用算术运算式隐式转换为数值 '12'-'5'

数据类型转换的案例：计算年龄

题面：要求在页面中弹出一个输入框，客户输入出生年份后，能使用弹出框显示年龄
解题：
var birthYear=prompt('请输入您的出生年份：');
alert("您的年龄是：" + (2022 - birthYear));

数据类型转换的案例：简单加法器

题面：要求在页面中弹出两个输入框，客户输入数字后，能使用弹出框显示结果
解题：
var num1=parseInt(prompt('请输入第一个相加的数字：'));
var num2=parseInt(prompt('请输入第二个相加的数字：'));
alert("计算结果是：" + (num1+num2));

其他类型转换为布尔型

方式 说明 举例
Boolean()函数 其他类型转为布尔型 Boolean('true')
代表空、否定的值会被转换为false，如''、NaN、null、undefined
其余值都会被转换为true
