循环的目的：在实际问题中，有许多具有规律性的重复操作，因此在程序中要完成这类操作就需要重复执行某些语句
在程序中，一组被重复执行的语句被称之为循环体，能否继续重复执行，取决于循环的终止条件。由循环体及循环的终止条件组成的语句，被称之为循环语句。
for的语法结构：
for(初始化变量;条件表达式;操作表达式)
{
循环体
}
断点测试是指自己在程序的某一行设置一个断点，调试时，程序运行到这一行就会停住，然后你可以一步一步往下调试，调试过程中可以看各个变量当前的值，出错的话，调试到出错的代码行即显示错误，停下。
断点测试：F12-->sources-->找到需要调试的文件-->在程序的某一行设置断点
watch:监视，通过watch可以监视变量的值的变化。
F11:程序单步执行，让程序一行一行的执行，这个时候，观察watch中变量的变化。
for循环可以输出相同的代码，可以循环输出不同的代码，也可以重复相同的操作

for循环案例：求1-100之间所有整数的累加和

var sum = 0;
for (i = 1; i <= 100; i++) { sum += i; }
alert("1-100之间所有整数的累加和是" + sum);

for循环案例：打印星星

题面：
第一行10⭐，第二行9⭐，以此类推。

for(i=0;i<10;i++){

}
while语句可以在条件表达式而真的前提下，循环执行指定的一段代码，直到表达式不为真时结束循环。
while 语句的语法结构如下：
while(条件表达式){
//循环体代码
}

while循环案例：打印人的一生，从1岁到100岁

题面：
从1岁到100岁
解题：
var age = 1;
while (age <= 100) {
console.log("今年" + age + "岁了！");
age++;
}

while循环案例：计算1~100之间所有整数的和

题面：
计算1~100之间所有整数的和
解题：
var num = 1, sum = 0;
while (num <= 100) {
sum += num;
num++;
}
console.log("1-100中所有的整数和是：" + sum);
do...while,该循环会先执行一次代码块，然后对条件表达式进行判断，如果条件为真，就会重复执行循环体，否则退出循环。
do...while 语句的语法结构如下：
do{
//循环体代码
}while(条件表达式)
do...while至少执行一次
